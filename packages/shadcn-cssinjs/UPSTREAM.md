# Upstream

The package structure and command architecture are intentionally based on the
MIT-licensed `packages/shadcn-cssinjs` package from
[`shadcn-ui/ui`](https://github.com/shadcn-ui/ui).

Initial upstream reference: `main` at `shadcn@4.13.0` (July 12, 2026).

CSS-in-JS engine setup and registry installation are maintained in this package.
