<h1 align="center">shadcn-cssinjs</h1>

shadcn-cssinjs is an unofficial community-led [StyleX](https://stylexjs.com) port of [shadcn/ui](https://ui.shadcn.com/).

> **Note** <br>
> **We are not affiliated with shadcn, but this project follows the same open-source philosophy.** <br>
> This is a project born out of the need for a shadcn/ui-like experience with CSS-in-JS instead of Tailwind.

Accessible and customizable components that you can copy and paste into your apps. Free. Open Source. **Use this to build your own component library.**

Built on top of [@base-ui/react](https://base-ui.com) (headless primitives) and styled with [@stylexjs/stylex](https://stylexjs.com).
