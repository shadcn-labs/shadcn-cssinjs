The Combobox is built by composing [Popover](/docs/components/popover) and [Command](/docs/components/command). There is no separate component
to install — add the building blocks and compose them.

## Installation

```bash
npx shadcn@latest add https://shadcn-cssinjs.vercel.app/r/popover.json
npx shadcn@latest add https://shadcn-cssinjs.vercel.app/r/command.json
```
