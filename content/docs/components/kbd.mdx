## Usage

```tsx
import { Kbd, KbdGroup } from "@/components/ui/kbd/kbd";
```

```tsx
<KbdGroup>
  <Kbd>⌘</Kbd>
  <Kbd>K</Kbd>
</KbdGroup>
```

## Composition

Use the following composition to build a `Kbd`:

```txt
KbdGroup
├── Kbd
└── Kbd
```
