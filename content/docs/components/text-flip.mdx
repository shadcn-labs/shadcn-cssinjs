## Usage

```tsx
import { TextFlip } from "@/components/ui/text-flip/text-flip";
```

```tsx
<TextFlip interval={1.5}>
  <span>StyleX</span>
  <span>atomic CSS</span>
  <span>CSS-in-JS</span>
</TextFlip>
```

Each child is shown in turn, flipping to the next after `interval` seconds.
