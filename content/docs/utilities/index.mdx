---
title: Utilities
description: Reusable StyleX utilities and hooks for common interface patterns.
---

Install the StyleX utility collection:

```bash
pnpm dlx shadcn@latest add @cssinjs/stylex-utils
```

The collection includes scroll fades, shimmer text, and a visually-hidden accessibility helper. The mobile breakpoint hook is available separately as `@cssinjs/use-mobile`.
