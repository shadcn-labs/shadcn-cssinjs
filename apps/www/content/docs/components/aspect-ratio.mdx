## Usage

```tsx
import { AspectRatio } from "@/components/ui/aspect-ratio/aspect-ratio";
```

```tsx
<AspectRatio ratio={16 / 9}>
  <img src="..." alt="Image" />
</AspectRatio>
```

## API Reference

### AspectRatio

The `AspectRatio` component displays content within a desired ratio.

| Prop        | Type     | Default | Required |
| ----------- | -------- | ------- | -------- |
| `ratio`     | `number` | `1`     | No       |
| `className` | `string` | -       | No       |
