## API Reference

### Badge

The `Badge` component displays a badge or a component that looks like a badge.

| Prop        | Type                                                     | Default     |
| ----------- | -------------------------------------------------------- | ----------- |
| `variant`   | `"default" \| "secondary" \| "destructive" \| "outline"` | `"default"` |
| `className` | `string`                                                 | -           |
